# Roadmap

## Termine

- base multi-projets .NET
- API locale
- Desktop WPF avec navigation MVP
- societes clientes
- exercices fiscaux
- declarations MVP
- dashboard basique
- audit log basique
- middleware d'erreurs
- premiers tests de services
- solution `DeclarationEmployerTunisie.sln`

## Partiellement termine

- dashboard : compteurs MVP, encore incomplets
- declarations : workflow MVP present, modele metier encore simplifie
- documentation : base pro ajoutee, a enrichir au fil des phases

## En cours

- stabilisation repository
- structuration documentaire

## Priorites suivantes

- authentification JWT et roles
- utilisateur courant et audit utilisateur
- finalisation Clients
- finalisation FiscalYears
- enrichissement du modele metier de declaration

## Limites restantes

- pas d'authentification complete
- pas d'import Excel fonctionnel
- pas de moteur fiscal actif
- pas de generation PDF
- pas d'archivage complet
- pas de backup applicatif finalise

## Prochaines versions

- `0.2.x` : securite et finalisation modules de base
- `0.3.x` : import Excel et controle fiscal
- `0.4.x` : exports internes structures et rapports
- `0.5.x` : archivage, maintenance et parametres
