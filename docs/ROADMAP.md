# Roadmap

## Termine

- base multi-projets .NET
- API locale
- Desktop WPF avec navigation MVP
- societes clientes
- exercices fiscaux
- declarations MVP
- dashboard basique
- audit log basique
- middleware d'erreurs
- premiers tests de services
- solution `DeclarationEmployerTunisie.sln`
- authentification JWT MVP
- utilisateurs et roles MVP
- modele metier declaration MVP
  - annexes
  - beneficiaires
  - lignes
  - anomalies
  - fichiers generes traces
  - archives tracees

## Partiellement termine

- dashboard : compteurs MVP, encore incomplets
- declarations : workflow MVP present, import et generation encore absents
- documentation : base pro ajoutee, a enrichir au fil des phases

## Priorites suivantes

- import Excel
- FiscalEngine
- generation d'exports internes structures
- rapports PDF
- archivage reel

- pas d'import Excel fonctionnel
