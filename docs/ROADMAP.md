- base multi-projets .NET

- Desktop WPF avec navigation MVP
- societes clientes
- exercices fiscaux
- declarations MVP
- dashboard basique
- audit log basique
- middleware d'erreurs
- premiers tests de services

- dashboard : compteurs MVP, encore incomplets

## Priorites suivantes

- rapports PDF de controle

- backup PostgreSQL
